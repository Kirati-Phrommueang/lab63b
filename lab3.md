# การทดลองที่3 การเขียนโปรแกรมเอาต์พุตสัญญาณดิจิดอล

## วัตถุประสงค์
1.ทดสอบการทำงานของโปรแกรมเอาต์พุตสัญญาณดิจิตอล

## อุปกรณ์* 
* 1.ไมโครคอนโทรเลอร์ ESP01
* 2.ซีเรียล
* 3.คอมพิวเตอร์
* 4.สายUSB
* 5.อะแดปเตอร์

## ศึกษาข้อมูลเบื้องต้น
* 1.ศึกษาข้อมูลของไมโครคอนโทรเลอร์ที่ใช้(https://docs.platformio.org/en/latest/boards/espressif8266/esp01_1m.html)
* 2.ศึกษาโค้ดของโปรแกรมที่ต้องการรันบนไมโครคอนโทรเลอร์(https://github.com/choompol-boonmee/lab63b/blob/master/examples/03_Output-Port/src/main.cpp)

## วิธีการทำการทดลอง
* 1.เชื่อมต่อสายUSBด้านหนึ่งเข้ากับคอมพิวเตอร์และอีกด้านเข้ากับซีเรียล
* 2.ต่ออะแดปเตอร์เข้ากับซีเรียล
* 3.ต่อไมโครคอนโทรเลอร์เข้ากับอะแดปเตอร์
* 4.เรียกโฟรเดอร์ที่เราบันทึกโปรแกรมเก็บไว้ โดยในการทดลองนี้โปรแกรมของเราอยู่ในโฟรเดอร์pattani โดยใช้คำสั่ง'cd pattani'ลงในCommand Prompt
* 5.เรียกโปรแกรมที่เราต้องการ โดยในการทดลองนี้เราจะใช้โปรแกรมที่1ในโฟรเดอร์pattani โดยใช้คำสั่ง'cd 03_Output-Port'ลงในCommand Prompt
* 6.รันโปรแกรมลงบนไมโครคอนโทรเลอร์ โดยใช้คำสั่ง'pio run -t uplond'ลงในCommand Prompt
* 7.ขณะรันโปรแกรมให้กดปุ่มสีดำก่อนแล้วกดปุ่มสีแดงบนซีเรียล
* 8.จากนั้นทดสอบโปรแกรมที่รันสำเร็จ โดยใช้คำสั่ง'pio device monitor'ลงในCommand Prompt


## การบันทึกผลการทดลอง
* จากการทดลองเมื่อไมโครคอนโทรเลอร์รันโปรแกรมสำเร็จ ได้ผลดังนี้
* ผลการทำงานของโปรแกรมที่รันสำเร็จ ได้ผลดังนี้
